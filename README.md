**Cybersecurity Portofolio**

About Me:
Saya adalah seorang lulusan muda yang sedang belajar dengan minat kuat pada bidang Cybersecurity, terbiasa menggunakan berbagai tools seperti Burp Suite, Wireshark, Kali Linux, dan Nmap untuk analisis serta pengujian keamanan. Saya memiliki dasar pemahaman pada topik-topik penting seperti malware, mobile application security analysis, data integrity failures, security misconfiguration, outdated components, broken access control, SSRF, cryptographic issues, session hacking & hijacking, network & wireless hacking, serta DoS/DDoS attacks.

Pengalaman Belajar:
1. Bootcamp Dibimbing            : Mendalami pembelajaran dan praktik yang berkaitan dengan Cybersecurity.
2. Platform TryHackMe & PicoCTF  : Aktif mengikuti challenge dan Capture the Flag untuk mengasah keterampilan eksploitasi dan mitigasi.
   
Proyek Akademik yang pernah dikerjakan:
1. DoS Attack dengan LOIC: Membuktikan dampak serangan dengan penemuan menurunnya kinerja sistem akibat traffic berlebihan pada sistem.
2. SQL Injection pada Form Login Dummy App: Demonstrasi kerentanan dan mitigasi dengan prepared statements terhadap sebuah Dummy Form Login.
3. Broken Access Control Analysis: Melakukan pembuktian Kelemahan Broken Access Control pada OWASP Juice Shop, menggunakan Burp Suite dan CVSS Calculator untuk mengidentifikasi, membuktikan, menilai, serta memberikan rekomendasi mitigasi.

Sebagai salah satu bentuk portofolio projek yang pernah saya kerjakan, saya melampirkan di bawah ini Laporan penugasan saya pada pengujian Broken Access Control.
